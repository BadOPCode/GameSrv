Descriptions

  1. SysOp Name
    • Your name, which is displayed/used in various parts of the system (in some drop files, the FROM string in the email verification message).
  2. SysOp Email Address
    • Your email address, which currently is only used in the FROM string for the email verification message.
  3. BBS Name
    • Your BBS Name, which is displayed/used in various parts of the system (in some drop files).
  4. First Node
    • The starting node number, used in conjunction with the Total Nodes setting to determine the range of node numbers to use.
  5. Total Nodes
    • The maximum number of nodes, used in conjunction with the Total Nodes setting to determine the range of node numbers to use.
      • Example 1: First Node 1 and Total Nodes 5 means the node range is 1,2,3,4,5
      • Example 2: First Node 5 and Total Nodes 3 means the node range is 5,6,7
